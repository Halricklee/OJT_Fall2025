Alien Dash – AWS-Powered Gaming Infrastructure

Project Proposal (Summary Version – Markdown)

1. Executive Summary

Alien Dash là game endless runner dùng Unity và AWS, giải quyết vấn đề sinh viên thiếu trải nghiệm tích hợp đa dịch vụ cloud. Dự án tích hợp Lambda, DynamoDB, API Gateway, Cognito và CloudFront để tạo game có avatar AI, 1v1 real-time racing và leaderboard động. Lợi ích gồm tăng khả năng thực hành cloud, tiết kiệm thời gian chuẩn bị giảng dạy, và nâng cao cơ hội việc làm. Thời gian thực hiện 12 tuần, chi phí $0–100, với 5 thành viên. Kết quả mong đợi: demo WebGL hoàn chỉnh, AI avatar, real-time mode và ≥90% test case đạt.

2. Problem Statement

Thị trường game phát triển mạnh nhưng PC endless runner còn thiếu, đa số tập trung mobile (≈90%). Người chơi thiếu lựa chọn trên desktop; game ít có personal avatar và thiếu chế độ real-time 1v1 → giảm retention đến 40%. Sinh viên thiếu dự án tích hợp cloud thực tế; mentor cần minh chứng kỹ thuật tốt. Không hành động sẽ bỏ lỡ thị trường casual PC gaming và cơ hội đào tạo cloud tích hợp. Cơ hội lớn với nhóm người chơi trẻ dùng laptop/PC (≈50%).

3. Solution Architecture

Hệ thống dùng kiến trúc serverless: CloudFront + S3 cho WebGL; API Gateway + Lambda cho logic; DynamoDB cho dữ liệu; Cognito cho auth; CodePipeline cho CI/CD. Luồng chính: User → Route53/WAF → CloudFront/S3 → API Gateway → Cognito → Lambda → DynamoDB → Streams/SNS → WebSocket. Bảo mật: WAF, IAM least-privilege, mã hóa at-rest & in-transit. Khả năng mở rộng dựa vào serverless auto-scale và CDN caching. Unity client và Admin Dashboard kết nối trực tiếp API Gateway.

4. Technical Implementation

Triển khai theo phases: (1) Setup core AWS; (2) Backend APIs; (3) Unity integration; (4) Test & optimize; (5) CI/CD deploy. Compute: Lambda; Storage: S3 + DynamoDB; Network: Route53, CloudFront, WAF. Quy trình Agile/Scrum, test bao gồm unit, integration, system, performance. Deployment tự động bằng CodePipeline/CodeBuild; rollback thông qua Lambda versioning và DynamoDB backups. Theo dõi bằng CloudWatch và X-Ray.

5. Timeline & Milestones

Week 1: GDD; Week 2–3: Architecture & data flow; Week 4–7: Core gameplay + leaderboard; Week 8–9: Boss + cutscene + shop; Week 10–11: Testing + optimization; Week 12: Deployment + demo. Milestones: GDD → Architecture → Gameplay → Boss → Testing → Demo. Nhóm 5 người phân nhiệm rõ ràng; dự phòng 1 tuần cho rủi ro kỹ thuật.

6. Budget Estimation

Dự án tận dụng AWS Free Tier; ước tính chi phí hẹp: $0–100 tổng trong 3 tháng nếu vượt Free Tier. Phần mềm & công cụ: Unity Personal, Python, Figma (miễn phí). Chi phí nhân lực không tính trực tiếp (academic project). Chiến lược tối ưu: nén assets, presigned uploads, theo dõi AWS Budgets để tránh overage.

7. Risk Assessment

Rủi ro chính: misconfiguration AWS, WebSocket instability, Unity–AWS integration issues, vượt Free Tier, team skill gaps, timeline delay, data exposure. Ưu tiên: cấu hình sai và WebSocket reliability. Giảm thiểu: dùng CloudFormation, reconnect + fallback HTTP, internal workshops, CloudWatch alerts và IAM chặt chẽ. Contingency: fallback leaderboard cục bộ, HTTP polling, chuyển sang tài khoản AWS Educate nếu cần.

8. Expected Outcomes

KPI ngắn hạn: API P95 <100ms, leaderboard latency <2s, uptime ≥99.5% trong demo, hỗ trợ 50+ concurrent, AI avatar xử lý <5s, chi phí vận hành < $30/tháng, ≥90% test case pass. Kết quả: WebGL demo hoạt động, tài liệu kỹ thuật hoàn chỉnh, học viên có portfolio, kiến trúc tái sử dụng cho khóa sau.

9. Game Design Overview – Alien Rush

Endless runner phong cách Sci-Fi, bối cảnh 3 vùng Việt Nam: Hanoi, Hoi An, Saigon; 15 level (5/region) với boss ở level 5/10/15. Hệ thống avatar: 2 avatar mặc định + AI-generated từ ảnh người chơi. Vật phẩm: buffs/debuffs, currency (₫10,000), shop mua item. Gameplay: dash/jump, obstacles dashable/non-dashable, mystery boxes, boss fights, local & global leaderboards, real-time 1v1 friend match với WebSocket sync và live events qua SNS/Streams.
